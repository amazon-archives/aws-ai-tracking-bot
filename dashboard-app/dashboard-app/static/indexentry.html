Welcome to the aws-ai-tracking-bot Sample Dashboard

There are two ways to access the dashboard:

  • Unauthenticated login allows the user to access the tracking-bot anonymously. Metrics are stored such that only data is stored unique to this device / browser. Other browsers and devices will not have access to this data. Access is relative to a user created for this browser / device combination.
  • Authenticated login requires the user to sign in to an account the user has created. If a user does not have an account, the user will need to create an account specifying a username, email address, and password. The benefit of this approach is that the user can access the dashboard from any device or browser. The user will login using their username / password combination and will see metrics entered from any device.
